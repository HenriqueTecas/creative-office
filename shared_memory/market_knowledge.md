### Data Center Infrastructure Management (DCIM)
- Schneider, Vertiv, Nlyte: established DCIM vendors with predictive failure capabilities
- Failure prediction accuracy: 60-70% for disk drives via SMART; near-random for power supplies and motherboards
- Last verified: 2026-03-09

### Warehouse Robotics
- Amazon Kiva, Ocado, Locus: centrally orchestrated swarm systems
- KION/Dematic: AGV vendors for mid-scale warehouses
- Slime mold algorithms: computationally trivial, implementable in ~50KB edge code
- Last verified: 2026-03-09
